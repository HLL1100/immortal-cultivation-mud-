# 修仙传 - 无限流MUD文字修仙放置手游

一款纯文字MUD风格的无限流修仙放置手游，主打离线挂机修炼、挖灵石、渡劫飞升等核心玩法。

## 游戏特色

- 🎮 **纯文字MUD风格** - 沉浸式修仙体验
- ⚡ **离线挂机修炼** - 关闭游戏也能获取修为
- ⛏️ **挖矿系统** - 挖掘灵石获取资源
- 🌟 **境界系统** - 从炼气期到飞升仙界的完整境界体系
- ⚡ **渡劫系统** - 突破大境界时需要渡过天劫
- 📜 **功法系统** - 学习功法提升修炼效率和战斗力
- 🎒 **背包系统** - 管理物品和资源
- 📋 **任务系统** - 完成任务获取丰厚奖励

## 本地运行

```bash
# 安装依赖
npm install

# 开发模式运行
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 部署到GitHub Pages

1. 将代码推送到GitHub仓库的 `main` 分支
2. 进入仓库的 Settings > Pages
3. 在 Source 下选择 "GitHub Actions"
4. 等待自动部署完成
5. 访问 `https://你的用户名.github.io/仓库名/`

## 游戏玩法

### 修炼系统
- 点击"开始修炼"自动获取修为
- 修为满后自动突破到下一层
- 关闭游戏后，下次打开可领取离线修为收益

### 挖矿系统
- 选择矿区开始挖矿
- 挖矿需要等待一定时间
- 获得灵石和挖矿经验
- 挖矿等级越高，收益越多

### 渡劫系统
- 达到境界圆满后需要渡劫
- 渡劫成功率受功法影响
- 失败会损失50%当前修为

### 功法系统
- 使用灵石升级功法
- 功法提升修炼速度、攻击力、防御力等
- 高级功法需要达到特定境界解锁
